Agent Configuration

Conversation Generation

Embedded Failure Patterns

  • Date Format Issues (3): Invalid formats like 03/15/2024 instead of 2024-03-15

Evaluation Strategy

  • LLM Stage: Deep semantic analysis only on conversations with issues (cost optimization)

Key Insights

  • Heuristics catch technical issues; LLM catches semantic/quality issues
  • Some conversations pass heuristics but fail LLM (e.g., travel_v1_issue_010)
  • Pattern clustering enables automatic proposal generation
